# GMS_ITProject
# 🏋️ Gym Management System

A full-stack Gym Management System designed to streamline gym operations, improve member experience, and simplify administrative workflows. This project includes a public-facing website and an admin management panel, connected through a Gym API that acts as a bridge between frontend and backend systems.

---

## 🌐 User Website Features

### 🏠 Homepage
The homepage serves as the main entry point of the website, providing an overview of the gym, quick navigation, and highlights of key services and offerings.

### 📋 Programs Page
Displays all available fitness programs offered by the gym, including:
- Strength Training
- Cardio Programs
- Weight Loss Programs
- Personal Training Sessions
- Group Fitness Classes

Users can browse and understand each program easily before enrolling.

### 🧑‍🏫 Trainers Page
Showcases gym trainers and highlights their expertise and classes. Each trainer profile may include:
- Specializations
- Experience
- Featured classes

### 💳 Membership & Subscription
Allows users to explore membership plans and subscribe based on their fitness goals. Features include:
- Monthly and yearly plans
- Subscription options
- Membership benefits overview

### ℹ️ About Us
Provides information about the gym, including:
- Mission and vision
- Background story
- Core values
- Facility overview

---

## 🔐 Admin Panel (Management System)

The admin dashboard is designed for staff and administrators to manage gym operations efficiently.

### 🔑 Login System
Secure authentication system for admin access.

### 👤 Member Management
- Search existing members
- Add new members
- Supports physical/walk-in registration
- Update or manage member details

### 🧑‍💼 Staff Management
- Register new staff members
- Manage staff roles and profiles
- Update staff information

### 🚪 System Controls
- Logout functionality
- Exit system button for secure session handling

---

## 🔗 Gym API Bridge
The Gym API acts as a bridge between the frontend website and backend system, ensuring:
- Seamless data communication
- Real-time updates for members and staff
- Secure handling of authentication and data requests

---

## ⚙️ Tech Stack
- Frontend: HTML, CSS, JavaScript (or React if applicable)
- Backend: Node.js / C#
- Database: MySQL / SSMS
- API: RESTful Gym API
