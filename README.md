# 🏋️ Gym Management System

A modern gym management website designed to streamline gym operations and improve member experience. This project includes membership management, trainer programs, subscription plans, authentication features, and a responsive user interface focused on clean design and usability.

---

## ✨ Features

### 🔐 Authentication System
- User login & registration
- Secure authentication flow
- Session management
- Role-based access (Admin / Member)

### 🏠 Homepage
- Modern landing page
- Hero section with call-to-action
- Featured training programs
- Gym highlights and services
- Responsive design for all devices

### 🧑‍🏫 Trainer Programs
- Display trainer profiles
- Workout and training plans
- Program descriptions and schedules
- Personalized coaching information

### 💳 Subscription & Membership
- Multiple membership tiers
- Monthly and yearly plans
- Subscription comparison section
- Membership management system

### 📋 Member Management
- Member registration system
- Attendance tracking
- Profile management
- Membership status monitoring

### ℹ️ About Us Section
- Gym mission and vision
- Company background
- Fitness philosophy
- Team introduction

### 📱 Responsive UI
- Mobile-friendly design
- Modern and clean interface
- Smooth animations and transitions
- Optimized user experience

## 🎯 Project Goals

The goal of this project is to create a complete gym management platform that simplifies administrative tasks while providing gym members with a seamless and engaging digital experience.

This system was built with a focus on:
- Clean UI/UX design
- Scalability
- User accessibility
- Real-world gym management workflows
